Latest Book

Oct 26 2004
So I haven't had any life changing events happen this week (which is probably a good thing). Actually, only one significant thing happened but I'll tell you about that later (huh?). One ordinary thing I did was read a book, a very good book in fact, though since its SciFi you probably have to like that stuff to enjoy it. Its been a while since I've read any fiction (excluding the multiple readings of LOTR in the past 4 or 5 years) and when a friend of mine handed me Ender's Game I figured it would be a good way to waste some time other than computer stuff, video games, and such. I'm not in the SciFi book circles so this book was new(ish) to me, but it seems like it is quite a popular book. Bestseller, awards, mentioned on Slashdot all the time, yadda, yadda. I think that the reason I enjoyed it so much was that it was not science fiction for science fictions sake only. Rather, it was a complelling story which just happens to be set in a futuristic world. It certainly does have aliens, anti-gravity, and space-ships, but more importantly it has Ender. Such a poor yet amazing little kid. The book really makes you feel for him. I wouldn't classifly the book as having super-deep themes and philosophical insights, but I also would not classify it as superficial. There is some good questions there about children, how we treat them, and how we shape them, yet for the most part it an interesting and engaging story. You can only read so many serious things right? Have some fun...
