Is programming writing?

Aug 30 2004
Computers and technology often seem to be inherently different than the arts, but is there really such a great divide between the two? Here is a paper I wrote begins to address this question. It looks at programming in comparison to writing, programming languages in comparison to languages, and computer code in comparison to literature. This paper turned out to be a bit more than a typical class project since I was quite inspired by some of the ideas found while researching the topic. So please take a look at it, especially if you are a techie who thinks the code they write is more than just a a block of computer instructions, or if you are a complete non-techie who doesn't understand how programming can be quite a creative and artistic task.
